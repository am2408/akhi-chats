# Akhi Chats

Akhi Chats is a modern chat application inspired by Discord, featuring real-time messaging, notifications, video calls, screen sharing, and file uploads. This application is built using Next.js and deployed on Vercel for optimal performance and scalability.

## Features

- **Real-time Chat**: Engage in conversations with friends and communities in real-time.
- **Notifications**: Stay updated with notifications for messages and activities.
- **Video Calls**: Connect with friends through high-quality video calls.
- **Screen Sharing**: Share your screen during calls for collaborative work.
- **File Uploads**: Easily upload and share files within chats.

## Tech Stack

- **Frontend**: React, Next.js, Tailwind CSS
- **Backend**: Next.js API routes
- **Database**: Prisma with a relational database
- **Real-time Communication**: WebSockets
- **Deployment**: Vercel

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/akhi-chats.git
   cd akhi-chats
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Set up environment variables:
   - Copy `.env.example` to `.env` and fill in the required values.

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:3000`.
